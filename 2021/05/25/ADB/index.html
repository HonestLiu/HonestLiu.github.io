什么adb啊

所述的Android调试桥（adb）是一个开发工具便于Android设备和个人计算机之间的通信。这种通信通常是通过USB电缆完成的，但也支持Wi-Fi连接。

adb就像Android开发的“瑞士军刀”。它提供了由命令详细描述的众多功能adb --help。下面的“流行的adb命令”部分中列出了一些更常用的命令。

安装adb和fastboot

Google托管的zip仅包含adb和fastboot。您可以按照以下说明进行设置。

在Windows上

  1. 从Google下载Windows zip。
  2. 将其提取到某处-例如， %USERPROFILE%\adb-fastboot
  3. 在Windows 7/8上：
    • 在桌面上，右键单击“我的电脑”，然后选择“属性”。
    • 在“系统属性”窗口中，单击“高级”选项卡
    • 在“高级”部分中，单击“环境变量”按钮
    • 在“环境变量”窗口中，突出显示“系统变量”部分中的“路径”变量，然后单击“编辑”按钮
    • 追加;%USERPROFILE%\adb-fastboot\platform-tools到现有路径定义的末尾（分号分隔每个路径条目）
  4. 在Windows 10上：
    • 打开“开始”菜单，然后键入“高级系统设置”
    • 选择“查看高级系统设置”
    • 单击高级选项卡
    • 打开“环境变量”窗口
    • 在“系统变量”下选择路径变量，然后单击“编辑”按钮
    • 点击“新建”按钮
    • %USERPROFILE%\adb-fastboot\platform-tools在文字栏位中插入
  5. 安装通用adb驱动程序，然后重新启动。

配置 adb

要adb与您的设备一起使用，您需要启用开发人员选项和USB调试：

  1. 打开设置，然后选择“关于”。
  2. 点击“内部版本号”七次。
  3. 返回，然后选择“开发人员选项”。
  4. 向下滚动，然后检查“调试”下的“ Android调试”或“ USB调试”条目。
  5. 将设备插入计算机。
  6. 在计算机上，打开终端/命令提示符，然后键入adb devices。
  7. 设备上应显示一个对话框，要求您允许进行USB调试。选中“始终允许”，然后选择“确定”。

恭喜你！adb现在可以与您的设备一起使用了。

配置 fastboot

使用fastboot（如果您的设备支持的话）应该只涉及重新启动到快速启动模式。在快速启动模式下，您可以键入fastboot devices以验证是否检测到您的设备。

热门adb命令

  • adb shell-在设备上启动外壳
  • adb push <local> <remote>-将文件推<local>送到<remote>
  • adb pull <remote> [<local>]-将文件拉<remote>至<local>。如果<local>未指定，它将拉到当前文件夹。
  • adb logcat-允许您实时查看设备日志。您可以使用adb logcat -b radio来查看广播日志，并adb logcat -C以彩色查看日志
  • adb install <file> -将给定的.apk文件安装到您的设备上
